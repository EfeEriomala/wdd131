Overview

This document outlines the site plan for TechWith-TK Academy, an online platform dedicated to providing coding courses, tech workshops, and resources for aspiring developers and career-switchers.

Site Name

TechWith-TK Academy - This name reflects the innovative and accessible approach to tech education, emphasizing skill-building and empowerment.

Site Purpose

The website will showcase coding courses, tech workshops, student projects, and enrollment options, while offering a user-friendly experience for students to explore and register for programs..

Scenarios

  • How can I learn in-demand tech skills like coding or cybersecurity?
  • How do I Regiser and get started?

Color Schema

Primary Color: Deep Blue (#0a07af) - Used for headings, navigation, and accents.

Accent Color: White(#FFF) - For text elements.

Background: sky blue (#12acdb) - Used for buttons and call-to-action to ensures a clean and elegant look.

Typography

  • Playfair Display - Used for headings and brand name.
  • Poppins - Used for body text.
  • Raleway - Used for buttons and call-to-action text.
